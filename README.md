# 📄 PDF Merger

A simple Python application that merges multiple PDF files into a single PDF document with an easy-to-use interface.

## Features

- Merge multiple PDFs
- Preserve document quality
- Fast processing
- Simple interface
- Cross-platform

## Tech Stack

- Python
- PyPDF2 (or pypdf)
- Tkinter (if GUI)

## Installation

bash
git clone https://github.com/yourusername/PDF-Merger.git

pip install -r requirements.txt

python app.py
Usage
Select PDF files.
Arrange the desired order.
Click Merge.
Save the merged PDF.
Future Improvements
Drag & Drop support
Split PDF
Rotate pages
Password-protected PDFs
